完成函数 createModule，调用之后满足如下要求：  
1、返回一个对象  
2、对象的 greeting 属性值等于 str1， name 属性值等于 str2  
3、对象存在一个 sayIt 方法，该方法返回的字符串为 greeting属性值 + ', ' + name属性值

``` js
function createModule(str1, str2) {
    function Fun(){
        this.greeting = str1;
        this.name = str2;
        this.sayIt = function(){return this.greeting + ', ' + this.name};
    }
    return new Fun();
}
```
